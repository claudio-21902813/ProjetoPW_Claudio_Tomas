Técnicas usadas

Carateristicas Elementos/técnicas Exemplos
Layouts Usou-se o CSS grid e CSS flexbox para construção de layouts O CSS Grid foi usado para a construção do layout da pagina, e o flexbox foi usado para organizar os elementos da forma que era suposto
Responsividade Foram usadas media queries que adaptam o layout para telemovel e PC consoante o tamanho da janela do browser. Para PC, foi usada este layout com um media querie no minimo de 601 px Para Mobile, foi usada este layout com um media querie no maximo de 600 px
Seletores Foram usados os tipos de seletores: universal, tipo, classe, id, pseudo-class, pseudo-elemento Na area de navegação, foi usado o :hover para quando o cursos passar por cima do elemento , ser possivel mudar a cor de fundo
Etiquetas HTML5 main, header, article, footer, figure,iframe,body, nav , section,article ,p,h,table,thead,tbody,form,fieldset,input,label,ul,li,a,aside,svg,image, html,figcaption,caption,legend,br,hr,picture Exemplos
Animações Foram feitas animações usando keyframes que permite "dar vida" a elementos no website Foi usado por exemplo, para dar um zoom no texto quando se passava o cursor, e tambem na pagina Inicial.
